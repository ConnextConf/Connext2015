Connext 2015
==================

与同龄的极客或科技「发烧友」在同一会场中分享自己的项目、产品、经验和极客生活。在 Connext 2015 新一代青少年科技大会上，让更多有共同梦想的人关注到你，和更多有共同兴趣的人结交朋友，向更多不同水平的科技人分享你的想法……

无论你是是有阅历和经验的技术宅，或是刚刚入门的初学者，你都能在 Connext 2015 这个线下公共平台获得新的学习机会和机遇。
